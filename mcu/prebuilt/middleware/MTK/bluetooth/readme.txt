Bluetooth library usage guide

Brief:          This module provides the header files and libraries of the Bluetooth stack.
Usage:          For the header files, please refer to the API referenece document in mcu\doc\.
                For the libraries, you do not directly call the libraries which are wrapped by mcu\middleware\bluetooth module. 
                For more information, please refer to mcu\middleware\MTK\bluetooth\readme.txt

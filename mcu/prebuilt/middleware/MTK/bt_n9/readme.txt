Bluetooth N9 patch usage guide

Brief:          This module is the Bluetooth N9 patch binary. It supports headset, earbuds and speaker project

Usage:          Move the corresponded bin to the final output folder before flash tool loading the config file. Usually, it's done by script.
                     
Dependency:     N/A

Notice:         1) For speaker mode, there are 3 binaries to be combined. It also done by script.

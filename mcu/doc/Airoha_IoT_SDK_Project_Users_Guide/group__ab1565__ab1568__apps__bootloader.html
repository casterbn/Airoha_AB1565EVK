Overview
  • This is not an example project. This project is a bootloader for AB1565_AB1568. The developer can refer to this project to understand how to build a bootloader.
Hardware and software environment
  • Supported platform
    • AB1565_AB1568 evaluation Kit (EVK).
Directory contents
  • Source and header files
    • main.c: Main program
    • inc/hal_feature_config.h: AB1565_AB1568's feature configuration file.
    • src/custom_fota.c: Fota update function customization.
    • GCC/startup_bootloader.s: AB1565_AB1568 startup file for GCC.
    • GCC/syscalls.c: The minimal implementation of the system calls.
  • Project configuration files using GCC
    • GCC/feature.mk: Feature configuration.
    • GCC/Makefile: Makefile.
    • GCC/bootloader.ld: Linker script.
Run the example
  • Connect board to the PC with serial port cable.
  • Build bootloader and an example project with the command, "./build.sh ab1565_ab1568_evk example_project_name bl" under the SDK root folder and download the binary file to AB1565_AB1568 EVK.
  • Power on AB1565_AB1568 EVK and bootloader log are showen on serial tool.

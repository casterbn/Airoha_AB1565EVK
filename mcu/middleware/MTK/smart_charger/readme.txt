Smart charger case module usage guide

Brief:          This module is the smart charger case interface. 
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/middleware/MTK/smart_charger/module.mk" in your GCC project Makefile.

Dependency:     To enable it, please define MTK_SMART_CHARGER_ENABLE to "y".

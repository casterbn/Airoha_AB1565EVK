ROFS module usage guide

Brief:          This module is the ROFS (read-only file system) interface.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/middleware/MTK/rofs/module.mk" in your GCC project Makefile.
                KEIL: Drag the middleware/MTK/rofs folder to your project. Add middleware/MTK/rofs/inc to include paths.
                IAR: Drag the middleware/MTK/rofs folder to your project. Add middleware/MTK/rofs/inc to include paths.
Dependency:     To enable it, please define MTK_PROMPT_SOUND_ENABLE to "y".
Notice:         None.
Relative doc:   None.
Example project:None.
